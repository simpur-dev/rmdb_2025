### 1.2 执行器依赖关系

```
                   ┌────────────────────┐
                   │   QlManager        │  (执行入口)
                   └──────────┬─────────┘
                              │
              ┌───────────────┴───────────────┐
              │                               │
     ┌────────┴────────┐            ┌─────────┴─────────┐
     │  DML Executors  │            │  Query Executors  │
     └────────┬────────┘            └─────────┬─────────┘
              │                               │
  ┌───────────┼───────────┐      ┌────────────┼────────────┐
  │           │           │      │            │            │
Insert     Delete      Update  SeqScan    IndexScan    NestedJoin
  │           │           │      │            │            │
  └───────────┴─────┬─────┘      └────────────┼────────────┘
                    │                         │
              RmFileHandle              [Projection]  [Sort]
              (+ 索引维护)                     │
                                              ▼
                                         子算子链（火山模型）
```

### 1.3 调用链示例：`SELECT * FROM t WHERE id = 1`

```
Parser → AST → Analyze → LogicalPlan
                           │
                           ▼
                       Optimizer（选择 IndexScan）
                           │
                           ▼
                   IndexScanExecutor::beginTuple()
                           │
                           ▼
                   IxIndexHandle::lower_bound(key)
                           │
                           ▼
                   BufferPoolManager::fetch_page(leaf_page)
                           │
                           ▼
                   DiskManager::read_page(...)
```
